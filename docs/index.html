METODOLOGIA

HIPÓTESE

Estamos interessados em avaliar se duas variáveis categóricas são independentes ou se existe associação entre elas. Assim, formalizamos:

\(\begin{cases} H_0: \text{As variáveis são independentes (não existe associação entre elas)} \\ H_1: \text{As variáveis são dependentes (existe associação entre elas).}\end{cases}\)

MÉTRICAS DE ASSOCIAÇÃO

  • O Teste qui-quadrado de Pearson é usado para verificar a associação entre as variáveis qualitativas e a satisfação com o serviço online.
  • O Teste Exato de Fisher é alternativa conservadora, para quando o Teste qui-quadrado não é adequado.
  • O coeficiente de Cramer foi utilizado para medir a força da associação entre as variáveis.

SUPOSIÇÕES

  • Em geral, supõe-se que cada da elemento amostra é independente.
  • Assume-se que a amostra é representativa da população.
  • Para o Teste Qui-quadrado, são exigidas frequências esperadas menores que 5.
  • Para o Teste de Fisher, supõe-se que os totais marginais são fixos.

ESTATÍSTICAS DE TESTE

TESTE QUI-QUADRADO DE INDEPENDÊNCIA

Cálculo da estatística

  1. Calcular as frequências esperadas para cada combinação de categorias, sob a hipótese nula de independência.
  2. Calcular a estatística do teste usando a fórmula:
    \(\displaystyle \chi^2 = \sum_{i=1}^n\sum_{j=1}^m \frac{(O_{ij} - E_{ij})^2}{E_{ij}}\), onde O é a frequência observada e E é a frequência esperada.

TESTE QUI-QUADRADO DE INDEPENDÊNCIA

Resíduos Padronizados

  • Os resíduos padronizados são dados pela fórmula:
    \(R_{ij} = \frac{O_{ij} - E_{ij}}{\sqrt{E_{ij}}}\)
  • Auxiliam na identificação do sentido da associação entre as categorias, indicando se a frequência observada é maior ou menor do que a esperada.

TESTE QUI-QUADRADO DE INDEPENDÊNCIA

Comentários

  • A frequência esperada corresponde ao valor que mantém a proporção entre as categorias, caso as variáveis sejam independentes.
  • A estatística de teste mede a discrepância entre as frequências observadas e esperadas
  • A ponderação pelo valor esperado visa evitar que categorias com maiores frequências tenham um impacto desproporcional no resultado do teste.

TESTE EXATO DE FISHER

Estrutura do teste

  • Dado o número de observações em cada categoria, o teste calcula a probabilidade de obter uma distribuição tão extrema quanto a observada, assumindo que as variáveis são independentes.
  • A estatística do teste é uma variável aleatória que segue uma distribuição hipergeométrica de parâmetros \(N\), \(K\), \(n\) e \(k\), onde:
    • \(N\): número total de observações
    • \(K\): número de observações na categoria de interesse
    • \(n\): número de observações na categoria de comparação
    • \(k\): número de observações na interseção das categorias Fórmula:
      \(p = \frac{\binom{K}{k} \binom{N-K}{n-k}}{\binom{N}{n}}\)
